Подвиг 4. Продолжите программу, в которой объявляется строка с именем str и максимальной длиной 100 символов (включая символ конца строки). Инициализируйте str строковым литералом "Best string!". Замените символ '!' на символ '.'.

P. S. В консоль ничего выводить не нужно.

#include <stdio.h>

int main(void)
{
    char str[100] = {"Best string!"};
    
    for (int i = 0; i < sizeof(str); i++){
        if (str[i] == '!')
            str[i] = '.';
        else 
            continue;
        }
    __ASSERT_TESTS__ // макроопределение для тестирования (не убирать и должно идти непосредственно перед return 0)
    return 0;
}
